## 完整操作流程

### 第一步：选择数据集

从主菜单选择 **"Start Training"** 或按 `Ctrl+T`，进入数据集选择页面。

1. **数据集1 — NEXRAD 3D 雷达 (.nc)**：点击 **Browse** 按钮，进入文件浏览器
   - 用 `↑` `↓` 箭头键导航
   - `Enter` 进入子目录或选择文件
   - `Backspace` 返回上级目录
   - `Ctrl+H` 回到用户主目录
   - `ESC` 取消选择
   - 选中后路径自动填入输入框

2. **数据集2 — 风暴轨迹 (.csv)**：同上操作

3. **预处理输出目录**：默认 `./processed`，可修改

4. 点击 **Validate (Ctrl+V)** 验证数据格式
   - 通过后显示绿色提示 "Validation passed"
   - 失败则显示具体错误信息

5. 点击 **Preprocess (Ctrl+P)** 开始预处理
   - 显示进度条
   - 完成后提示 "Preprocessing complete"

6. 点击 **Skip (Ctrl+S)** 跳过预处理，进入下一步

### 第二步：配置训练参数

进入训练配置页面后，可设置以下参数：

#### 数据参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| Spatial Window | 空间窗口经纬度范围 | 2.56 |
| Grid Size | 网格分辨率 H×W | 128 |
| Vertical Layers | 垂直层数，留空=全部29层 | (空) |
| Refl. Normalization | 反射率归一化方式 | Min-Max |

#### 模型参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| Encoder Depth | 编码器深度 | 4 |
| Base Channels | 基础通道数 | 64 |
| Diffusion Coef | 可学习扩散系数 | 0.1 |

#### 预训练权重（可选）
- 点击 **Browse** 选择已有的 .pt 或 .pth 权重文件
- 支持两种格式：本工具保存的完整 checkpoint，或纯 state_dict 文件
- **Load Optimizer State**：是否恢复优化器状态（继续训练时开启）
- **Strict Param Match**：是否严格匹配参数名（迁移学习时可关闭）

#### 训练参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| Run Name | 训练任务名称，留空使用时间戳 | (空) |
| Output Root Dir | 输出根目录 | ./output |
| Batch Size | 批次大小 | 2 |
| Epochs | 训练轮数 | 100 |
| Learning Rate | 学习率 | 0.001 |
| Early Stop Patience | 早停耐心值 | 15 |
| GPU Acceleration | GPU 加速开关 | 开启 |
| Mixed Precision | 混合精度训练 AMP | 开启 |

点击 **Start Training (Ctrl+T)** 开始训练。

### 第三步：监控训练

进入训练监控页面：

- **左侧面板**：训练进度条、实时指标（Loss/Accuracy/Precision/Recall）、GPU 状态
- **右侧面板**：实时日志流
- **底部按钮**：
  - **Pause (Ctrl+P)**：暂停训练
  - **Resume (Ctrl+R)**：恢复训练
  - **Stop (Ctrl+X)**：终止训练
  - **Background (Ctrl+B)**：切换到后台运行，返回主菜单

#### 后台运行模式

点击 **Background (Ctrl+B)** 后：
1. 训练在后台线程继续运行
2. 返回主菜单，可进行其他操作（如查看历史记录）
3. 再次点击 **Start Training** 可重新进入监控界面
4. 系统自动检测到正在运行的训练并恢复监控

#### 训练输出结构

所有结果保存在 `output/<任务名称>/` 目录下：

```
output/
└── 20260505_143022/          # 时间戳（或自定义名称）
    ├── checkpoints/           # 模型权重 (best_model.pt, final_checkpoint.pt)
    ├── logs/                  # 日志文件
    │   └── training.log       # 训练日志
    ├── config.yaml            # 配置副本
    ├── metrics.csv            # 指标 CSV
    └── plots/                 # 训练效果图
        ├── loss_curve.png
        ├── accuracy_curve.png
        └── vorticity_sample.png
```

### 第四步：模型推理

从主菜单选择 **"Model Inference"** 或按 `Ctrl+I`，进入推理页面。

1. **Checkpoint 权重文件**：选择训练好的 .pt 或 .pth 文件
2. **推理参数**：
   - **Prediction Steps**：预测步数，默认 36 步（每步 5 分钟 = 3 小时）
   - **Time Step**：时间步长（秒），默认 300
   - **Longitude Range**：经度范围，如 `262.0,272.0`，留空自动
   - **Latitude Range**：纬度范围，如 `29.0,39.0`，留空自动
   - **Vertical Layer Range**：垂直层范围，如 `0,29`，留空全部
   - **Use GPU**：GPU 推理开关
3. **输入数据**：选择 .npz 文件或目录（目录=批量预测）
4. **输出目录**：默认 `./predictions`
5. 点击 **Run Inference (Ctrl+R)** 开始推理
6. 底部日志区域显示进度和结果

## 常见问题

**Q: 数据验证失败？**
检查 .nc 文件是否包含 `index`、`Reflectivity`、`AzShear` 变量。检查 .csv 文件是否包含 `Time`、`Longitude`、`Latitude` 列。

**Q: 训练中途想退出？**
点击 **Background (Ctrl+B)** 切换到后台，或点击 **Stop (Ctrl+X)** 优雅终止。直接关闭终端会丢失当前 epoch 进度。

**Q: 如何从之前的 checkpoint 继续训练？**
在训练配置页面的 "Pretrained Weights" 中选择之前的 checkpoint，勾选 "Load Optimizer State"。

**Q: .pt 和 .pth 有什么区别？**
没有区别，都是 PyTorch 的模型权重文件。本工具统一输出 .pt，但加载时同时兼容 .pt 和 .pth。
